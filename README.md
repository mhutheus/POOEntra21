# Exercícios de Programação Orientada a Objetos (POO)

Este repositório reúne os exercícios desenvolvidos durante a disciplina de Programação Orientada a Objetos (POO) do curso Entra21.

## Tecnologias utilizadas

- Java
- Visual Studio Code
- Git e GitHub

## Estrutura do projeto

Os exercícios estão organizados na pasta `exercicios`, sendo cada pasta referente a uma atividade proposta durante as aulas.

## Objetivo

Praticar os principais conceitos de Programação Orientada a Objetos, como:

- Classes e objetos
- Encapsulamento
- Métodos
- Atributos
- Relacionamento entre classes
- Manipulação de objetos
